## 📖 Sobre o Curso

O curso é uma jornada completa pelo ecossistema da linguagem mais utilizada no planeta. Patrocinado pelo **Google**, o conteúdo é estruturado para levar o aluno desde os primeiros comandos até recursos modernos do ECMAScript, com foco total na lógica e na prática.

> "Não basta apenas decorar comandos, é preciso entender como pensar como um programador." — Gustavo Guanabara

### Por que aprender JavaScript?
O JavaScript está em todo lugar: sites, aplicativos, sistemas e servidores. É a base fundamental para quem deseja trabalhar com frameworks como **React, Vue.js, Angular** ou atuar no backend com **Node.js**.

---

## 🧠 O que foi abordado?

O aprendizado foi dividido em seis módulos graduais e sólidos:

* **Fundamentos:** Primeiros scripts, tipos de dados e variáveis.
* **DOM (Document Object Model):** Manipulação de elementos HTML e CSS via JS para criar páginas dinâmicas.
* **Condicionais:** Estruturas de decisão (`if/else`, `switch`).
* **Repetições:** Laços de controle (`while`, `do while`, `for`).
* **Variáveis Compostas:** Uso de Arrays (Vetores).
* **Funções:** Modularização e reutilização de código.
* **Práticas Modernas:** Alinhamento com as normas atuais do **ECMAScript**.

Para realizar este curso, utilizei um ambiente de desenvolvimento moderno e leve:

## 🚀 Como utilizar este repositório

1.  **Clone o projeto:**
    ```bash
    git clone [https://github.com/cmigueldev/curso_JavaScript.git](https://github.com/cmigueldev/curso_JavaScript.git)
    ```
2.  **Navegue pelas pastas:**
    Os arquivos estão organizados por módulos ou aulas, conforme a evolução do curso.
3.  **Execute os códigos:**
    * Para arquivos web: Abra os arquivos `.html` no seu navegador.
    * Para scripts puros: Utilize o Node.js no terminal:
        ```bash
        node nome-do-arquivo.js
        ```
